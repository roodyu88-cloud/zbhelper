# ZBHelper

ZBHelper — Первый БЕСПЛАТНЫЙ и быстрый хелпер по законодательной базе любого из серверов проекта с интеграцией ИИ поиска прямо у вас на компьюетере во время игры, созданное как интерактивный справочник законодательной базы на GTA 5 RP. Оно позволяет быстро находить нужные статьи, делать заметки и использовать чат с ИИ ассистентом для более простого поиска нужной вам информации.

## ⚡ Основные возможности
- **Законодательная база:** Структурированная база законами с мгновенным локальным поиском по кодексам и статьям.
- **AI-ассистент:** Встроенный чат на базе искусственного интеллекта (работает через бесплатный Groq API), который помогает анализировать ситуации и трактовать статьи.
- **Плавающие заметки:** Полупрозрачное окно для ведения заметок, которое может отображаться поверх игры или других окон с настраиваемой прозрачностью.
- **Избранное:** Быстрый доступ к самым важным кодексам и конкретным статьям в один клик.
- **Фоновая работа:** Приложение сворачивается в системный трей и мгновенно вызывается по горячей клавише.

## ♾️ Установка и запуск
Приложение поставляется в виде полностью автономного `.exe` файла. Для работы **не требуется** устанавливать Python или дополнительные библиотеки.

1. Перейдите в раздел [Releases](https://github.com/roodyu88-cloud/zbhelper/releases) на GitHub.
2. Скачайте актуальный файл `ZBHelper_Setup.exe`.
3. Установите и запустите приложение как обычную программу для Windows.

## 🛡️ Информация о безопасности (VirusTotal)
При проверке установщика `ZBHelper_Setup.exe` на VirusTotal пара малоизвестных антивирусов может выдавать предупреждения. Это **ложные срабатывания (False Positives)**, которые очень часто встречаются у программ, написанных на Python.

**Почему это происходит:**
1. **Специфика PyInstaller:** Сборка приложения выполнена с помощью PyInstaller. При запуске он распаковывает необходимые для работы библиотеки во временную папку `Temp` в Windows. Такое поведение иногда триггерит параноидальные эвристические алгоритмы некоторых антивирусов.
2. **Отсутствие цифровой подписи:** Программа не имеет дорогостоящего коммерческого сертификата разработчика (Code Signing Certificate), поэтому система Windows (SmartScreen) и антивирусы могут относиться к файлу с недоверием как к "неизвестному издателю".

*Весь исходный код приложения открыт и находится в этом репозитории. Вы в любой момент можете изучить его и убедиться в полной безопасности, а при желании — собрать `.exe` самостоятельно!*
