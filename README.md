# ZBHelper

ZBHelper — это легковесное и удобное десктопное приложение, созданное как интерактивный справочник законодательной базы (ориентировано на сервера GTA 5 RP). Оно позволяет быстро находить нужные статьи, делать заметки и получать консультации с помощью встроенного ИИ-ассистента.

## Основные возможности
- **Интерактивный справочник:** Структурированная база законодательства с мгновенным локальным поиском по кодексам и статьям.
- **AI-ассистент:** Встроенный чат на базе искусственного интеллекта (работает через Groq API), который помогает анализировать ситуации и трактовать статьи.
- **Плавающие заметки (Overlay):** Полупрозрачное окно для ведения заметок, которое может отображаться поверх игры или других окон с настраиваемой прозрачностью.
- **Избранное:** Быстрый доступ к самым важным кодексам и конкретным статьям в один клик.
- **Фоновая работа:** Приложение сворачивается в системный трей и мгновенно вызывается по горячей клавише.

## Установка и запуск
Приложение поставляется в виде полностью автономного `.exe` файла. Для работы **не требуется** устанавливать Python или дополнительные библиотеки.

1. Перейдите в раздел [Releases](https://github.com/roodyu88-cloud/zbhelper/releases) на GitHub.
2. Скачайте актуальный файл `ZBHelper_Setup.exe`.
3. Установите и запустите приложение как обычную программу для Windows.

## Стек технологий

- **Интерфейс (Frontend):** HTML, CSS, JavaScript (Vanilla)
