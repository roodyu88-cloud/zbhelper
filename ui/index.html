Настройка AI (Groq API)

Для работы AI-ассистента требуется бесплатный токен от Groq:

  1. Перейдите на console.groq.com/keys
  2. Авторизуйтесь или создайте аккаунт
  3. Нажмите кнопку Create API Key
  4. Скопируйте ключ (начинается на gsk_) и вставьте его в настройки ZBHelper
Важное примечание: Сервис Groq может блокировать подключения из РФ. Если ИИ выдает ошибку "Forbidden" или "Ошибка 403", пожалуйста, включите VPN перед использованием AI.
